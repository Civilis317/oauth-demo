# OAuth2 demo

Trying out OAuth2 authentication for a Spring Boot application with Thymeleaf UI.


### Reference Documentation
Mandatory env vars:
* GITHUB_CLIENT_ID
* GITHUB_CLIENT_SECRET

Optional env vars:
* LOGLEVEL_SPRING
* LOGLEVEL_APPLICATION
